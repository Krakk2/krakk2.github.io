Главная Как пользоваться?

Что это и как этим пользоваться?

На сайте установлен скрипт, позволяющий получать изображения, которые делают пользователи программ для скриншотов.

Зачем это нужно?

В первую очередь для поиска конфиденциальных данных пользователей,
которые делают скриншоты своих паспортных данных, номеров телефонов,
банковских карт, а так же электронной почты (иногда даже с паролями),
примеры вы можете наблюдать ниже.

А также любители поискать личные фото так же найдут для себя что-то интересное.


Как пользоваться?

На главной странице Нашего сайта есть поле, в которые Вам следует вставить URL скриншота для парсинга в формате https://prnt.sc/m3a7rc

По умолчанию поле уже имеет значение для парсинга - это говорит что Вам достаточно нажать на кнопку Go!

После того как будет спарсен скриншот, Вы можете использовать кнопки (P)rev (Предыдущий скриншот) и (N)ext (Следующий скриншот) для навигации по скриншотам.

Кнопка Random :) представляет собой, случайное спарсивание скриншота.

Почему так долго грузит?

Скриншоты грузятся в режиме реального времени, тысячами пользователей в полном размере без сжатия.
Есть скриншоты по 100Кб, а есть и по 10-20Мб.

Парсер не загружает скриншоты, что делать?

Попробуйте нажать сочетание клавишь Ctrl+F5 для обновления с очисткой кэша.
